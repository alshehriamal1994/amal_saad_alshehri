## Reproducibility

This repository provides evaluation and analysis code only.

### Dataset

The dataset used in this work is available from the Durham University Research Data Repository:

- DOI: http://doi.org/10.15128/r14x51hj064
- Landing page: https://collections.durham.ac.uk/files/r14x51hj064

### Reproduction Workflow 
1. Download the dataset from the DOI landing page.
2. Generate candidate passages using BM25 or dense retrieval.
3. Apply neural reranking models using the evaluation scripts.
4. Compute evaluation metrics from TSV outputs.
