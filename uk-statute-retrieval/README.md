# Neural Reranking for UK Statutory Retrieval

This repository contains code for candidate generation, neural reranking, and evaluation
used in the paper **Neural Reranking for UK Statutory Retrieval**.

## Data Access Statement

Our supporting research dataset is published in the **Durham University Research Data Repository**.

- DOI: http://doi.org/10.15128/r14x51hj064  
- Landing page: https://collections.durham.ac.uk/files/r14x51hj064

This GitHub repository contains **code only**. No dataset files are included.

## Repository Structure

- `scripts/daily_scraper.py` (optional): helper to fetch/update external resources; not required to reproduce paper results.
